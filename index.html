Agentic Zoning Validation — Integrated Frontend & Backend View

Unified visualization of the Agentic Zoning Validation Agent (UI) and the LLM → ArcGIS Request Pipeline (Backend).

Diagram Layer Role in System
Left — UI: Agentic Zoning Validation Agent Frontend (User-facing Layer) User interacts in natural language (“Is this parcel compliant within Zone B?”). The UI sends structured queries and displays map results (PASS / VIOLATION).
Right — LLM → ArcGIS Request Pipeline Backend (Reasoning + GIS Execution Pipeline) Converts user intent into ArcGIS geoprocessing requests, executes validation logic, and returns structured responses + map updates back to the UI.
